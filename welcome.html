Buoyant Widget Library for Axure RP 8

Welcome to Wave's widget library for Axure. Have a look around!


To get started with Axure, download the latest version and install it.


To start using this widget library click on the menu icon in Libraries and load from Axure share, use code VED5MY.


Alternatively, you can load this from GDrive by navigating to design > assets > Axure Widget Library > buoyant-v0.1.rplib.

To use icons, you'll need to  add them as a web-font. Follow these instructions:

1. Install the wv-icons.ttf font from GDrive.

2. Add web fonts to your project by clicking on share and then Edit. Click on Web Fonts and add the following snippet of code. Name it as wv-icons


font-family:"wv-icons";

font-weight:normal;

font-style:normal;

src:url("https://s3.amazonaws.com/wave-buoyant/2.2.6/fonts/wv-icons.woff") format("woff")


3. Create a Font Mapping by adding icomoon against wv-icons.
